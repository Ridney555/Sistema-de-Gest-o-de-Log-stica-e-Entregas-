CAA - trabalho pratico funcionalidades do projeto (Grafos/filas)
---------------------------------------------------------------------------------------------------
=> Sistema de de Gestão de Logística e Entregas:

 - utilização da estrutura de dados Grafos para representar as zonas ou os pontos de entregas, Filas para pedidos com prioridade como por exemplo entregas com urgência vao sair primeiro.

 - Utilizacao de CRUD para permitir ao utilizador criar novos pontos de entrega ou novas estradas  mais rapidos, listar os pontos disponiveis, atualizar a rota se o caminho    antigo     estiver interditado assim procurando por um um novo caminho mais rapido, consiguir remover um ponro de entrega.

 - Utilizacao de base de dados MySQL para salvar o histrorico e o mapa.
   Leitura de todos os dados de forma automatica, quando iniciar e fechar;
   Gerar relatorios;

 - Uso do Algoritmo de Dijkstra em que o sistema nao ira apenas guardar os dados, mas tambem calcular a melhor rota encontrar o caminho mais curto.
