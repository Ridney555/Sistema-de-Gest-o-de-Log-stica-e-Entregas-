CAA - trabalho pratico funcionalidades do projeto (Grafos/filas)
---------------------------------------------------------------------------------------------------
1. Gestão da Rede (Grafo)
-> Vértices e Arestas - Cadastro de localidades e das estradas que as ligam.
-> Pesos - Definição de distâncias e tempos entre pontos.
-> Mapa Dinâmico - Visualização e edição da malha logística.

1. Gestão de Frota (CRUD)
-> Controlo de Veículos - Cadastro, listagem, edição e remoção de veículos (matrícula, modelo e carga).
-> Gestão de Status - Monitorização de disponibilidade (Livre, Em Rota, Manutenção).

1. Operações e Lógica (Estruturas de Dados)
-> Fila de Prioridade - Organização de encomendas por urgência.
-> Otimização de Rota: Uso do Algoritmo de Dijkstra para calcular o caminho mais curto no Grafo.

1. Base de dados MySQL
-> Base de Dados - Armazenamento permanente de todos os dados no MySQL.
-> Relatórios - Gerar  ficheiros (PDF) com o resumo das entregas e estatísticas.
